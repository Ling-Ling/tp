#Download

##With Git
```
git clone git@github.com:iliaskarim/tp.git
```


##Without Git
```
mkdir tp
cd tp
curl https://nodeload.github.com/iliaskarim/tp/zipball/master --O tp.zip
unzip -j tp.zip
rm tp.zip
```


#Launch

```
chuck --loop tp &
chuck + wub
```

OR  

```
chmod 755 l
```

THEN  

```
./l wub
```


#Terminate

```
chuck --kill
```

OR  

```
killall chuck
```
