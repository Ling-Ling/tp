#Download

##With Git
```
git clone git@github.com:iliaskarim/tp.git
```


##Without Git
```
mkdir tp
cd tp
curl https://nodeload.github.com/iliaskarim/tp/zipball/master --O tp.zip
unzip -j tp.zip
rm tp.zip
```


#Launch

```
chuck --loop tp &
chuck + wub
```

OR  

```
chmod 755 l
```

THEN  

```
./l wub
```


#Terminate

```
chuck --kill
```

OR  

```
killall chuck
```


#Perform

##Master

```
mkdir tp
cd tp
curl https://nodeload.github.com/Ling-Ling/tp/zipball/master --O tp.zip
unzip -j tp.zip
rm tp.zip
chmod 755 l
./l master_bell
```


##Slave
```
mkdir tp
cd tp
curl https://nodeload.github.com/Ling-Ling/tp/zipball/master --O tp.zip
unzip -j tp.zip
rm tp.zip
chmod 755 bell
./bell $
(where $ is the first letter of the name of the computer)
```
